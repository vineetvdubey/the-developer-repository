Get all developers

    GET /api/developers
    Sample Response Body:
    [{
      "id": "gcnit",
      "avatar_url": "https://avatars.githubusercontent.com/u/4833751?v=4"
    }, {
      "id": "sagarjain0907",
      "avatar_url": "https://avatars.githubusercontent.com/u/20463272?v=4"
    }]
    Status: 200

Add a developer

    POST /api/developers
    Sample Request Body:
    {
      "github_id": "gcnit",
      "linkedin_id": "gcnit",
      "codechef_id": "gc_nit",
      "hackerrank_id": "gcnit",
      "twitter_id": "gc_nit",
      "medium_id": "gc_nit"
    }
    Sample Response Body:
    {
      "id": "gcnit"
    }
    Status: 201 (User Created), 400 (GitHub username is invalid)

    DELETE /api/developers/:id
    Status: 204 (Deleted)
